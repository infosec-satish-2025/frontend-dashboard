# Frontend Dashboard

A React-based admin dashboard for monitoring our microservices.

## Features

- Real-time service health monitoring
- User analytics
- API usage statistics
- System alerts

## Setup

```bash
npm install
npm start
```

## Environment Variables

Create a `.env` file:

```
REACT_APP_API_URL=http://localhost:3000
```

## Tech Stack

- React 18
- Chart.js for visualizations
- Axios for API calls
- React Router for navigation
